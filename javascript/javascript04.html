01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 창조하는데 사용합니다. 변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다.
let 키워드를 사용하여 선언됩니다. 각 변수는 선언된 후에 특정 값을 저장할 수 있고, 이 값은 나중에 참조하거나 변경할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        console.log(x);
        console.log(y);
        console.log(z);
}
결과 확인하기
100
200
