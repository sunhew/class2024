01. .at()

"문자열".at(위치값)

  • 지정한 숫자(index)를 받아, 문자열에서 행당되는 인덱스의 요소를 반환합니다.
  • 대소문자를 구별합니다.
  • 문열의 길이보다 큰 인덱스를 지정하면 undefiend를 반환합니다.
  • 음수인 경우 문자열의 오른쪽에서부터 인덱스를 셉니다.
{
        "javascript reference".at()       //j
        "javascript reference".at(0)      //j
        "javascript reference".at(1)      //a
        "javascript reference".at(2)      //v
        "javascript reference".at(3)      //a
        "javascript reference".at(4)      //s
        "javascript reference".at(5)      //c
        "javascript reference".at(-1)     //e
        "javascript reference".at(-2)     //c
        "javascript reference".at(-3)     //n
        "javascript reference".at(-4)     //e
        "javascript reference".at(-5)     //r
        "javascript reference".at(100)    // undefined
}

06. .includes()

"문자열".includes(위치값)

  • 특정 문자열이 다른 문자열 내에 포함되어 있는지 여부를 확인합니다.
  • 문자열이 포함되어 있으면 true를, 그렇지 않으면 false를 반환합니다.
  • 문열의 길이보다 큰 인덱스를 지정하면 undefiend를 반환합니다.
  • 대소문자를 구별합니다.
{
        let str = "javascript reference";

        console.log(str.includes("reference")); // true
        console.log(str.includes("JavaScript")); // false (대소문자 구분)
        console.log(str.includes("reference", 10)); // false (검색 시작 위치가 10이므로)
}

07. .indexOf()

"문자열".indexOf(찾고자하는 문자열, [위치값])

  • 찾고자 하는 문자열이 발견되면 해당 인덱스를 반환하고, 발견되지 않으면 -1을 반환합니다.
  • 문자열 내에서 특정 문자열의 첫 번째 인덱스를 반환합니다.
  • 특정 문자열이 포함되어 있는지 여부를 확인하는 데 사용됩니다.
  • 대소문자를 구별합니다.
{
        let str = "Hello, world!";

        console.log(str.indexOf("world")); // 7
        console.log(str.indexOf("hello")); // -1 (대소문자 구분)
        console.log(str.indexOf("l", 3)); // 3 (검색 시작 위치가 3이므로)
}

10. .match()

"문자열".match(검색할 정규식 패턴)

  • 문자열에서 정규식과 일치하는 부분을 찾아 배열로 반환합니다.
  • 정규식과 일치하는 부분을 배열로 반환합니다. 일치하는 부분이 없으면 null을 반환합니다.
  • 정규식에 /g 플래그를 사용하면 모든 일치 항목을 반환합니다.
  • 배열의 첫 번째 요소는 전체 일치 문자열이며, 이후 요소는 캡처 그룹에 해당하는 부분 문자열입니다.
{
        let str = "The quick brown fox jumps over the lazy dog.";

        console.log(str.match(/quick/)); // ["quick"]
        console.log(str.match(/\w+/g)); // ["The", "quick", "brown", "fox", "jumps", "over", "the", "lazy", "dog"]
        console.log(str.match(/hello/)); // null
}

16. .replace()

"문자열".replace(regexp|substr, newSubstr|function)

    * regexp|substr: 교체할 대상이 되는 문자열 또는 정규식 패턴
    * newSubstr|function: 교체할 새로운 문자열 또는 교체 로직을 포함한 함수
  • 문자열에서 특정 패턴을 찾아 다른 문자열로 교체하여 새로운 문자열을 반환합니다.
  • 첫 번째 일치 항목만 교체하며, 정규식에 /g 플래그를 사용하면 모든 일치 항목을 교체할 수 있습니다.
  • 교체 문자열에 $1, $2 등의 특수 문자를 사용하면 캡처 그룹을 참조할 수 있습니다.
  • 교체 함수를 사용하면 더 복잡한 교체 로직을 구현할 수 있습니다.
{
        let str = "The quick brown fox jumps over the lazy dog.";
        
        console.log(str.replace("fox", "cat")); // "The quick brown cat jumps over the lazy dog."
        console.log(str.replace(/\w+/, "animal")); // "animal quick brown fox jumps over the lazy dog."
        console.log(str.replace(/\w+/g, (match) => match.toUpperCase())); // "THE QUICK BROWN FOX JUMPS OVER THE LAZY DOG."
}

18. .search()

"문자열".search("검색에 사용할 정규 표현식 객체")

  • 일치하는 문자열이 발견되면, 일치하는 부분의 시작 인덱스(0 이상의 정수)를 반환합니다.
  • 일치하는 문자열이 없으면 -1을 반환합니다.
  • 전역 검색 플래그(g)를 무시합니다. 즉, 정규 표현식에 전역 검색 플래그(g)가 있어도, 첫 번째 일치하는 결과만 찾고 검색을 중단합니다.
  • 호출된 문자열을 변경하지 않습니다. 즉, 이 메서드는 순수 함수(pure function)의 특성을 가집니다.
{
        var str = "Hello, world!";
        var n = str.search(/world/);
        console.log(n); // 7 출력
}

19. .slice()

"문자열".slice(추출 시작점의 인덱스[, 추출을 종료할 인덱스])

  • 두 개의 매개변수를 받음
  • 첫 번째 매개변수는 추출이 시작되는 인덱스를 지정하고, 두 번째 매개변수는 추출이 끝나는 인덱스를 지정합니다.
  • 두 번째 매개변수는 선택사항이며, 제공되지 않을 경우 문자열의 끝까지 추출합니다.
  • 추출을 종료할 인덱스에 해당하는 문자는 추출되지 않습니다. 
{
        var str = "Hello, world!";
        console.log(str.slice(7, 12)); // "world"
        console.log(str.slice(-6, -1)); // "world"
        console.log(str.slice(7)); // "world!"
        console.log(str.slice(-6)); // "world!"
}
  • 첫 번째 예제에서는 인덱스 7부터 12 전까지의 문자열인 "world"를 추출
  • 두 번째 예제에서는 문자열 끝에서부터 6번째 위치부터 끝에서 1번째 위치 전까지인 "world"를 추출
  • 세 번째 예제에서는 인덱스 7부터 문자열의 끝까지인 "world!"를 추출
  • 네 번째 예제에서는 문자열 끝에서부터 6번째 위치부터 문자열의 끝까지인 "world!"를 추출

20. .substring()

"문자열".substring(indexStart[, indexEnd])

    * indexStart: 추출을 시작할 0부터 시작하는 인덱스입니다. 만약 indexStart가 indexEnd보다 크면, substring은 두 매개변수를 자동으로 교환하여 처리합니다.
    * indexEnd: 추출을 종료할 인덱스로, 이 위치의 문자는 포함되지 않습니다. 이 매개변수는 선택적이며, 제공되지 않을 경우 문자열의 끝까지 추출됩니다. indexEnd가 문자열의 길이보다 큰 경우, 문자열의 끝이 추출의 종료점이 됩니다.
  • 두 개의 매개변수를 받을 수 있습니다.
  • 원본 문자열은 변경하지 않고, 지정된 부분만을 포함하는 새로운 문자열을 반환합니다.
  • 두 번째 매개변수는 선택사항이며, 제공되지 않을 경우 문자열의 끝까지 추출합니다.
  • 음수 인덱스를 허용하지 않습니다. 음수 값은 0으로 취급됩니다.
{
        var str = "Hello, world!";
        var result = str.substring(7, 12);
        console.log(result); // "world" 출력
}
