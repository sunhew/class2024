01. if문

if문은 조건문 중 하나로, 주어진 조건이 참(true)인지 거짓(false)인지에 따라 코드 블록의 실행 여부를 결정합니다. JavaScript에서는 다양한 값이 참 또는 거짓으로 평가될 수 있는 "진리값(Truthy)"과 "거짓값(Falsy)" 개념이 있습니다. 

{
        if (2) {
            console.log("실행되었습니다.(true)")
        } else {
            console.log("실행되었습니다.(false)")
        }
}

#거짓값(Falsy)이 되는 경우
*false *0 (숫자 0) *"" 또는 '' (빈 문자열) *null *undefined *NaN (Not a Number)
